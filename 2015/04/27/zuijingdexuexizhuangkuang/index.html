最近的学习状况

这段时间(具体是拿到腾讯实习offer后的这段时间)心慢慢静了下来，也开始了一些新的学习。这段时间比较偏于Android UI方面的学习，就动画这一块，知乎上有一个问答我觉得可以推荐下Github 比较好的开源项目是如何做的？。

自从上学期看了代码家的AndroidSwipeLayout这个库的Demo，深深得被其美丽的效果折服。自那以后，心中常想着什么时候我自己也能写一个这么Powerful的库。但上学期的时间被项目和后面的各种考试拖着，这学期开学后的时间又拿去准备实习的事情，所以一直没能好好的研究。这段时间空了下来，然后认真研究了下。

总体

代码家的这个库，按照他自己的目标，他有这么几个优点:

When I start to make this library, I set some goals:

Can be easily integrated in anywhere, ListView, GridView, ViewGroup etc.

Can receive onOpen,onClose,onUpdate callbacks.

Can notifiy the hidden children how much they have shown.

Can be nested each other.

Can handle complicate situation, just like this.

具体

思想

一个SwipeLayout中只有一个surfaceView，可以有多个(最多四个)bottomView。定义了四个滑动方向Left，Top，Right，Bottom。
